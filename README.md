# Total TypeScript Course

Personal workspace for the Total TypeScript materials.

Instead of keeping each workshop in a separate repo, I keep them here so I can move between topics a bit more easily.

## Structure

- `01-beginnes-typescript` - beginner TypeScript exercises
- `02-pro-essentials-workshop` - everyday TypeScript patterns and tooling
- `03-advanced-patterns-workshop-main` - advanced type patterns
- `04-type-transformations-workshop-main` - mapped types and type transformations
- `05-typescript-generics-workshop-main` - generics, inference, and overloads
- `06-react-typescript-tutorial` - React + TypeScript exercises

## Working on a module

Each folder is a standalone project with its own dependencies and scripts.

```bash
cd 06-react-typescript-tutorial
npm install
npm run exercise
```

Most exercises are split into `*.problem.*` and `*.solution.*` files.
